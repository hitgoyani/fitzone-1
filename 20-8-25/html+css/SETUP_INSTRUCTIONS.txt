SETUP ON NEW DEVICE:
====================

STEP 1: Install XAMPP
- Download XAMPP from: https://www.apachefriends.org/
- Install it on your computer
- Start Apache and MySQL (both should be green)

STEP 2: Setup Website
- Copy the entire "html+css" folder to: C:\xampp\htdocs\
- Your website will be at: http://localhost/html+css/

STEP 3: Setup Database
- Open phpMyAdmin: http://localhost/phpmyadmin
- Click "New" and create database named "mydata"
- Click on "mydata" database
- Click "SQL" tab
- Copy and paste the contents of "database/database.sql" file
- Click "Go"

STEP 4: Test Database Connection
- Test: http://localhost/html+css/php/db.php
- Should show: "Database connected successfully!"

STEP 5: Test Main Database Page
- View: http://localhost/html+css/php/index.php
- Should display your gym members and trainers data

PHP VALIDATION FIX:
==================
If you get PHP validation errors in your editor:

For VS Code:
1. Install PHP extension: "PHP IntelliSense" by Felix Becker
2. The .vscode/settings.json file is already configured
3. Restart VS Code to apply settings

For other editors:
- Set PHP executable path to your XAMPP PHP installation
- Usually located at: C:\xampp\php\php.exe

ALTERNATIVE: Use Online PHP Validator
- Visit: https://www.w3schools.com/php/phptryit.asp?filename=tryphp_compiler
- Copy and paste your PHP code to test

TROUBLESHOOTING:
================
- If database connection fails: Check if MySQL is running
- If website doesn't load: Check if Apache is running
- If tables don't exist: Import database/database.sql again
- If PHP validation errors: Configure PHP executable path
- If images don't load: Check that images are in the images/ folder
- If CSS doesn't load: Check that style.css is in the css/ folder
